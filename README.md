   - Policy Only
   - Permissive

   - Demonstrative

5. **Barriers & Needs**: What prevents you from doing more with AI in class?
   - Lack of access to paid tools
   - Uncertainty about academic integrity policies
   - Need for training/examples
   - Time constraints
